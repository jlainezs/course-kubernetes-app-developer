# Installation/setup

It is required:

- Kubernetes cluster
- kubectl installed

## Installation

```
$ curl -s "https://raw.githubusercontent.com/kubernetes-sigs/kustomize/master/hack/install_kustomize.sh"  | bash
```
To check it is properly installed run the command

```
$ kustomize version --short
```
